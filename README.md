# Github에 코드 업로드하기

1. 비주얼 스튜디오 코드 초기화하고 원하는 폴더만 다시 넣기!
2. 터미널 누르고 git init 입력 폴더 보기 숨긴 항목 보기 체크!
3. .git 폴더 생성됨
4. 파일 올리기
```
5. git add . 입력
```
